# Классификация токсичных комментариев для «Викишоп»

## Описание проекта
В интернет-магазине «Викишоп» пользователи могут оставлять комментарии, которые иногда содержат токсичные высказывания. Необходимо создать модель машинного обучения, которая автоматически классифицирует комментарии как токсичные или нетоксичные, чтобы ускорить и упростить модерацию.

**Цель** — построить модель с метрикой F1 не ниже 0.75 для надежного выявления токсичных комментариев.

---

## Данные
Датасет `toxic_comments.csv` содержит:
- **text** — текст комментария пользователя;
- **toxic** — целевой признак:
  - 0 — комментарий не токсичный;
  - 1 — комментарий токсичный.

---

## Этапы работы
1. **Загрузка и подготовка данных**
   - Загрузка датасета.
   - Очистка текста: удаление стоп-слов, приведение к нижнему регистру, токенизация, лемматизация.
2. **Анализ данных**
   - Распределение целевого признака.
   - Статистика по длине и структуре комментариев.
3. **Обучение моделей**
   - Применение различных алгоритмов машинного обучения.
   - Подбор гиперпараметров.
   - Оценка по метрике F1.
4. **Выбор лучшей модели**
   - Модель, показавшая F1 ≥ 0.75 на валидации.
5. **Тестирование**
   - Оценка на тестовой выборке.
6. **Выводы**
   - Итоговые результаты и рекомендации.
